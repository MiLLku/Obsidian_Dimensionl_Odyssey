- ### 목록
	- #### 구매
		- 위쪽 Panel 에 Buy Button을 눌러 구매를 진행할 수 있다.
		- 아이템 종류별로 "삼각형" Button을 누르면 목록이 펼쳐지고 접혀진다.
		- 펼쳐진 목록에는 해당 종류별 모든 아이템 이름이 적혀있다.
		- 원하는 아이템 이름을 누르면, 가운데 큰 Panel에 판매중인 아이템 Panel이 나열되고, 아이템 Panel에는  [아이템 이름/아이템 판매자/가격/구매(Button)]이 배열되어 있다.
		- 혹은 Search에 원하는 아이템 이름을 입력해 판매 정보를 찾을 수 있다.
		- Buy 버튼을 누르면 Details Panel이 띄워진 후, x1 Button 을 눌러 원하는 개수를 정한 후 Buy Button을 눌러 구매 완료한다.
		- 목록이나 판매 리스트는 Slider Bar를 드래그하거나, 마우스 휠을 돌려 변경 가능.
	- #### 판매
		- 위쪽 Panel 에 Sell Button을 눌러 판매를 진행할 수 있다.
		- 아이템 종류별로 "삼각형" Button을 누르면 목록이 펼쳐지고 접혀진다.
		- 펼쳐진 목록에는 해당 종류별 모든 아이템 이름이 적혀있다.
		- 원하는 아이템 종류를 누르면, 가운데 큰 Panel에 창고나 인벤토리에 존재하는 해당 아이템이 띄워진다. (선택하지 않을 시 모든 아이템이 2번째 그림처럼 띄워진다.)
		- 혹은 Search에 원하는 아이템 이름을 입력해 판매정보를 찾을 수 있다.
		- 판매를 원하는 아이템 슬롯을 클릭하면, Details Panel이 띄워진 후 Price를 입력하고 x1 Button을 눌러 개수를 정한 후 Sell Button을 누르면 거래소에 아이템이 등록된다.
