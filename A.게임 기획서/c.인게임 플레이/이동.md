- ### 리그오브레전드의 이동 매커니즘을 기반
	- ##### 걷기
		- 마우스 우클릭을 한 위치에 장애물을 고려해 최단거리로 이동
		- [[Noise Distance]] : 10M
		- [[Noise]] : 
	- ##### 뛰기
		- 이동 위치가 지정된 상태에서 Shift를 누르면 빠르게 이동
		- [[Noise Distance]] : 20M
		- [[Noise]] : 
	- ##### 조용히 걷기
		- 이동 위치가 지정된 상태에서 Alt를 누르면 조용히 이동
		- [[Noise Distance]] : 5M
		- [[Noise]] : 
	- ##### 제자리 서기
		- 이동 중 키보드의 S 버튼을 누르면 제자리에 정지
	- ##### 밀려남
		- 특정 스킬에 타격당하면 발생
		- 어느것도 고려하지 않고 스킬의 밀어내기 범위만큼 강제이동
		- 밀려나는 도중에는 [[기절]] 상태이상과 동일 취급
	- ##### 돌진
		- 특정 스킬을 사용하면 발생
		- 어느것도 고려하지 않고 스킬의 이동 범위만큼 특정 속도로 이동
		- [[Invincibleity]] 상태가 아니므로 [[Crowd Control]] 기술 등에 타격당하면 취소된 후 해당 기술의 효과가 발현된다.
