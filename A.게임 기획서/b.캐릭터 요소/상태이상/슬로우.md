해당 캐릭터의 이동 속도가 감소한다. 적용시키는 스킬마다 슬로우의 수치와 지속시간이 정해져있다.

| 명칭               | 영문 명칭       | 해설                                                                                                                                                                            | 비고 |
| ------------------ | --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---- |
| 슬로우 상태        | isSlowed        | 대상이 현재 슬로우 상태인가 아닌가 판단하는 bool 변수                                                                                                                           |      |
| 슬로우 수치        | slowFigure      | 슬로우의 정도를 정하는 수치. int 변수를 사용한다.                                                                                                                               |      |
| 슬로우 지속시간    | slowDuration    | 슬로우가 지속되는 시간. float 변수를 사용한다.                                                                                                                                  |      |
| 애니메이터         | animator        | 슬로우 애니메이션을 관리                                                                                                                                                        |      |
| 슬로우 코루틴      | SlowCoroutine() | 슬로우 지속을 관리, slowDuration 시간이 지날 경우 RemoveFaint()를 실행                                                                                                          |      |
| 슬로우 코루틴 참조 | slowCoroutine   | 코루틴 실행용 참조                                                                                                                                                              |      |
| 슬로우 적용        | ApplySlow()     | 이미 새로운 슬로우에 적용되었을 시, 기존에 적용된 슬로우 수치가 더 높을 경우, 남은 시간을 마저 적용시킨 후 그 시간을 빼준 새로운 슬로우 지속시간을 적용, 슬로우 애니메이션 관리 |      |
| 슬로우 제거        | RemoveFaint()   | slowDuration이 완전이 지난다면 이동속도가 원래 상태에 돌아옴.                                                                                                                   |      |
| 기존 이동속도      | chrMoveSpeed    | 슬로우가 적용되기 전 원래 캐릭터의 이동 속도를 저장.                                                                                                                                                                                |      |
