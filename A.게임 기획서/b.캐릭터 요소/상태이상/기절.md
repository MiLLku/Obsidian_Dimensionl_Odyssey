해당 캐릭터를 즉시 제자리에 멈추게 한 후, 행동 불가 상태에 빠뜨리는 것

| 명칭             | 영문 명칭        | 해설                                                                        | 비고 |
| ---------------- | ---------------- | --------------------------------------------------------------------------- | ---- |
| 기절 상태        | isFainted        | 대상이 현재 기절 상태인가 아닌가 판단하는 bool 변수.                        |      |
| 기절 지속 시간   | faintDuration    | 기절 상태이상이 지속되는 시간.                                              |      |
| 애니메이터       | animator         | 기절 애니매이션을 관리       |      |
| 기절 코루틴      | FaintCoroutine() | 기절 지속을 관리, faintDuration 시간이 지날 경우 RemoveFaint() 실행         |      |
| 기절 코루틴 참조 | faintCoroutine   | 코루틴 실행용 참조.                                                         |      |
| 기절 적용        | ApplyFaint()     | 이미 기절 상태일 시 기존 코루틴 중지, 그 후 새로운 지속시간으로 코루틴 적용, 기절 애니메이션 관리 |      |
| 기절 제거        | RemoveFaint()    | 기절 애니메이션에서 벗어남, 새로운 입력을 받을 수 있음                      |      |
