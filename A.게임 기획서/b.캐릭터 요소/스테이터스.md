| 명칭           | 영문 명칭             | 해설                                                                                                                                                            | 비고 |     |     |
| -------------- | --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---- | --- | --- |
| 체력           | Health                | 캐릭터의 체력. 캐릭터의 생사를 정하는 능력치로 30% 이하로 감소하면 "위급" 상태에 빠지며, 0%에 달하면 사망한다.                                                  |      |     |     |
| 정신력         | Mental                | 캐릭터의 정신력. LOL의 "기력" 시스템과 유사. 기본 스킬을 사용하거나 특정 도구를 사용할 때 감소한다. 0인 상태가 오래 지속되면 정신력의 자연회복 속도가 감소한다. |      |     |     |
| 물리 데미지    | Phisical Damage       | 물리 데미지 계수를 갖는 스킬의 데미지를 결정해주는 능력치이자 기본 공격의 데미지를 증가시켜주는 스텟.                                                           |      |     |     |
| 마법 데미지    | Magic Damage          | 마법 데미지 계수를 갖는 스킬의 데미지를 결정해주는 능력치. 기본 공격의 데미지에 관여하지 않기 때문에 물리 데미지보다 좀 더 높게 보정을 받는다.                  |      |     |     |
| 방어력         | Armor                 | 물리 데미지를 입히는 스킬 밑 기본 공격의 데미지를 감소시켜주는 능력치.                                                                                          |      |     |     |
| 저항력         | Resistance            | 마법 데미지를 입히는 스킬의 데미지를 감소시켜주는 능력치.                                                                                                       |      |     |     |
| 방어 관통력    | Armor Peneration      | 방어력의 일부를 무시하는 능력치.                                                                                                                                |      |     |     |
| 저항 관통력    | Resistance Peneration | 저항력의 일부를 무시하는 능력치.                                                                                                                                |      |     |     |
| 이동 속도      | Move Speed            | 캐릭터의 시간당 이동거리를 증가시켜주는 능력치. 이 게임에서 매우 신중하게 설정해야 할 능력치이다.                                                               |      |     |     |
| 치명타 확률    | Critical Chance       | 기본 공격이 치명타로 적중될 확률.                                                                                                                               |      |     |     |
| 치명타 데미지  | Critical Damage       | 치명타로 적용된 공격의 데미지 증가 배수.                                                                                                                        |      |     |     |
| 공격 속도      | Attack Speed          | 첫번째 기본 공격에서 두번째 기본 공격까지의 간격.                                                                                                               |      |     |     |
| 상태 이상 저항 | Abnormal Tolerance    | 수치에 따라서 상태 이상에서 벗어나는 속도가 달라진다.                                                                                                           |      |     |     |
| 사정거리       | Attack Range          | 캐릭터의 기본 공격이 가능한 범위. 캐릭터의 특정 스킬에 의해 변화할 수 있다.                                                                                                                                                                |      |     |     |
| 정신력 재생    | Mental Regeneration   | 정신력이 자동회복되는 속도. 영향을 줄 수 있는것은 맵의 특정 오브젝트, 회복 아이템 부류, 아티팩트 등에 있다.                                                     |      |     |     |
| 체력 재생      | Health Regeneration   | 체력이 자동회복되는 속도. 인조인간 정신의체의 등급에 따라서 속도가 다르다.                                                                                      |      |     |     |
| 스텔스 수치    | Stelth Figures        | 캐릭터의 소음이 발산되는 거리. 특정 캐릭터의 스킬이나 아티팩트에 의해서 변할 수 있다.                                                                           |      |     |     |
