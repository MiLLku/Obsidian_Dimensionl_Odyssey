짧은 시간 내에 효과를 볼 수 있고, 강력한 효과를 가지고 있는 대신, 부작용도 존재한다.

| 명칭             | 영문 명칭       | 설명                        | 비고 |
| ---------------- | --------------- | --------------------------- | ---- |
| 아이템 설명      | itemDescription | 아이템 설명이 들어감        |      |
| 아이템 이름      | itemName        | 아이템 이름이 들어감        |      |
| 행동 시간        | actionTime      | 효과 발현까지 행동하는 시간 |      |
| 지속 시간        | effectDuration  | 효과가 지속되는 시간        |      |
| 구매 가격        | purchasePrice   | 상인에게서 구매하는 가격    |      |
| 판매 가격        | salePrice       | 상인에게 판매되는 가격      |      |
| 아이콘           | icon            | 아이템 표시 스프라이트      |      |
| 아이템 효과 적용 | applyEffect()   | 효과 적용하는 함수                            |      |
